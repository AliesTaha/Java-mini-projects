<img src="https://img.shields.io/badge/java-%23ED8B00.svg?style=for-the-badge&logo=java&logoColor=white"/>

## Running The Game
1. Download the game files from the releases page or clone the repository.
2. Navigate to the game directory in your terminal.
3. Compile the game with the following command

```bash
javac Main.java
java Main
```
